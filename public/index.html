🎥 Agora Token Server

Welcome to the Agora Token Server API. This service generates secure tokens for Agora RTC (Real-Time Communication) sessions.

GET /api/ping

Health check endpoint to verify the server is running.

Response: { "message": "pong" }

GET /api/rtc

Generate an Agora RTC token for joining a channel.

Query Parameters:

  • channel (required) - Channel name
  • uid (required) - User ID (positive integer)
  • role (optional) - User role: 'publisher', 'host', or 'audience' (default: 'audience')
  • expiry (optional) - Token expiry in seconds: 60-86400 (default: 3600)

Example:

/api/rtc?channel=myChannel&uid=1234&role=publisher&expiry=3600
